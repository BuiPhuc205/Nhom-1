# Nhom-1
Tổng quan hệ thống
Chức năng:

Hiển thị tổng số sinh viên trong hệ thống

Thống kê điểm GPA trung bình

Cung cấp cái nhìn tổng quát cho quản trị viên (admin)

Giúp admin nhanh chóng nắm được tình trạng dữ liệu sinh viên

- Mục đích: Hỗ trợ quản lý và theo dõi tổng thể hệ thống.

Quản lý sinh viên

Chức năng:

Thêm mới sinh viên (mã sinh viên, tên, GPA)

Hiển thị danh sách sinh viên

Xóa sinh viên theo mã

Quản lý thông tin học tập cơ bản của sinh viên

-Mục đích: Thực hiện các nghiệp vụ chính liên quan đến sinh viên.

 Lưu trữ dữ liệu
 
Chức năng:

Lưu danh sách sinh viên ra file CSV

Đọc dữ liệu sinh viên từ file CSV khi khởi động chương trình

Đảm bảo dữ liệu không bị mất khi chương trình kết thúc

- Mục đích: Tách riêng xử lý lưu trữ khỏi nghiệp vụ để dễ bảo trì.

Đăng nhập – Đăng ký

Chức năng:

Đăng ký tài khoản người dùng mới

Đăng nhập hệ thống bằng tài khoản đã đăng ký

Kiểm tra tính hợp lệ của thông tin đăng nhập

Phân biệt người dùng trước khi truy cập hệ thống quản lý

- Mục đích: Bảo mật hệ thống và kiểm soát quyền truy cập
